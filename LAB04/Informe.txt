Conclusiones:
-	En este laboratorio aprendimos a instalar un proyecto de express utilizando npm, con el comando ---save
	para guardar la dependencia en el package.json del proyecto.
-	Utilizamos Postman para probar los distintos métodos HTTP del proyecto y mostrar diferentes resultados 		para cada uno de ellos.
-	Aprendimos a manejar las rutas del navegador desde nuestra  index.js en nuestra aplicación de express y 	utilizamos diferentes recursos como imagenes, css y scripts con  "app.use".
-	Manejamos los errores que pueden presentarse al momento de ingresar a la página, esto sucede cuando 	el estado del request es 4XX para recursos no encontrados, y 5XX para errores en el servidor.
-	Utilizamos nuestros conocimietos sobre CSS para mejorar la apariencia de la landing-page de nuestro
	proyecto con estilos y animaciones, sin el uso de Bootstrap o plantillas.
